This documents are the frond end app.

1.- You need to copy this documentos into their own folder (just like this one that they are inside)
2.- Then you need to open the folder into your Code editing terminal, and then Run  % npm install.
3.- Now that your data for the npm node modules is installed in the folder you can run the back end first before starting the front end.
4.- Once the back end server is runing, you can run %npm start for the front end script.
